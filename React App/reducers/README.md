Command to installing

> npm i react-redux

2.For installing Redux from npm command is

> npm i redux

3.For installing redux logger command is

> npm i redux-logger

4.For installing redux promise

> npm i redux-promise

5.For installing redux thunk

> npm install --save redux-thunk
